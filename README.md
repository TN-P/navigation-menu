# navigation-menu

An interactive and responsive navigation menu built using **HTML, CSS, and JavaScript**.  
This project demonstrates modern frontend features such as scroll tracking, animations, dark mode, and accessibility.

---

##  Live Demo  
https://tn-p.github.io/navigation-menu/

---

##  Features

- Responsive navigation bar with hamburger menu  
- Smooth scrolling between sections  
- Active link highlighting  
- Scroll progress indicator  
- Navbar background change on scroll  
- Section reveal animations (IntersectionObserver API)  
- Light/Dark theme toggle (saved in localStorage)  
- Scroll-to-top button  
- Keyboard navigation using arrow keys  
- Accessible navigation with ARIA roles  

---

##  Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)
